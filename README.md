# Next.js Lab

This repository serves as a personal lab for experimenting with Next.js features and various related technologies.

## Current Labs

### Better Auth

A Next.js application demonstrating an improved authentication flow.

## Getting Started

Each lab within this repository is an independent Next.js project. To get started with a specific lab, navigate to its directory and follow its respective `README.md` instructions (if available) or standard Next.js development practices.

For example, to run the `better-auth` lab:

```bash
cd better-auth
npm install # or yarn install, pnpm install, bun install
npm run dev # or yarn dev, pnpm dev, bun dev
```
